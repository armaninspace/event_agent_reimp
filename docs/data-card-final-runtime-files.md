# Data Card: Final CSV Runtime Files For Reimplementation

This data card identifies the final CSV data files needed to run the event-agent research system without rebuilding raw joins or manipulating source data.

## Intended Use

These files support a reimplementation that starts from the final prepared runtime artifacts:

- city-week event/economic tests
- MSA-week event/economic tests
- CSV-backed semantic/query layers
- business evidence reports
- playback/regression runs

No raw data rebuild is required when these files are present.

## Required Files

| File | Required | Purpose | Current Size | Rows |
| --- | --- | --- | ---: | ---: |
| `data/reference/joined_city_week_game_economic.csv` | yes | Final city-week game/economic mart for locality-level matched tests and EDA. | 3,037,004 bytes | 5,777 |
| `data/reference/joined_msa_week_game_economic.csv` | yes | Final MSA-week game/economic mart for MSA matched tests and semantic queries. | 16,997,114 bytes | 48,723 |

## Copy Checklist

For a final-file-only reimplementation, copy exactly:

```text
data/reference/joined_city_week_game_economic.csv
data/reference/joined_msa_week_game_economic.csv
```

Optional:

```text
data/reference/real_weather_city_week.csv
```

Do not copy raw seed files unless you plan to rebuild the final marts.
Do not copy `semantic_layer.sqlite`; if the new implementation wants SQL, it should load/query the CSVs directly or build its own local cache.
Do not copy `questions.json`; the new implementation should generate questions.

## Validation Checklist

After copying the files:

1. Confirm the two required CSV files exist.
2. Confirm `joined_city_week_game_economic.csv` has at least one row and includes `cityid`, `week_start_monday`, `revenue_all`, `merchants_all`, `has_game`, and `game_count`.
3. Confirm `joined_msa_week_game_economic.csv` has at least one row and includes `msa_code`, `week_start_monday`, `revenue_all`, `merchants_all`, `has_game`, `game_count`, and `msa_block_id`.
4. Run a one-turn smoke workflow before running a 20-turn regression.
